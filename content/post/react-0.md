# React คืออะไร  
React ก็คือจาวาสคริปต์ไลบรารีสำหรับสร้างส่วนติดต่อกับผู้ใช้ ว่าง่ายๆก็คือตัวหน้าเว็บนั่นเอง 
โดยที่ React มีจุดเด่นอยู่ 3 ข้อหลักๆ คือ  
1.  Declarative  
2.  Component-Based  
3.  Learn Once, Wrtie Anywhere
